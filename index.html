About

Who I Am & What I Do

I'm Sivasree Govindaraj, a Software Engineer with nearly 1.5 years of experience in QA Automation and Software Development. I enjoy building reliable automation solutions and writing clean, maintainable code that improves software quality. My work focuses on ensuring applications perform as expected before they reach users, while continuously exploring better ways to automate testing and development workflows.

My experience includes building UI and API automation frameworks using Java, Playwright, Selenium WebDriver, Rest Assured, Postman, SQL, and Spring Boot. I've worked on enterprise applications across banking and financial domains, contributing to automation frameworks, API validation, regression testing, defect analysis, and Agile development. I enjoy solving complex testing challenges through automation while continuously improving engineering efficiency.

Beyond my day-to-day work, I enjoy building tools that solve real problems. One of my projects, CogniFill, is a Chrome Extension built with Playwright that scans DOM elements and automates repetitive form interactions using a Spring Boot backend. I'm also exploring AI-powered testing and workflow automation with n8n to create smarter, faster, and more efficient testing solutions.

API Monitoring & Automated Test Reporting
n8n · Google Sheets · HTTP Nodes · Gmail Integration

Scheduled n8n workflows that ping APIs at regular intervals, validate status codes, log every result to Google Sheets automatically, and fire email alerts on failures — so the team knows before any user does.

  • Runs on scheduled trigger nodes that poll each endpoint at a fixed interval, no manual checks needed.
  • Validates response status codes and timing against expected thresholds before logging a result.
  • Failures fire an instant Gmail alert, so the team hears about a problem within minutes, not at the next manual pass.
  • Every run appends to a shared Google Sheet, giving a lightweight uptime history without a dedicated dashboard.
